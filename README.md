# r-devcontainer-template
A VS Code Dev Containers template with R

## Quickstart guide
```
code .
```

* If the **Reopen in Container** popup shows up in the bottom-right corner then simply press the respective button.
* Otherwise press CTRL+SHIFT+P for the command runner to appear and select **Dev Containers: Open Folder in Container...**

When you open a terminal it should immediately show a prompt with which one can interact with the container, e.g.

```
rstudio@6bf8f99bf129:~/workspace$
```
